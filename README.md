# LoginChallengeFirebase

- Simple App for login using Firebase, Facebook Services and save data in Firebase Realtime Data Base

## Technologies 🛠️

- [Swift]
- [Swift Package Manager]
- [Firebase Auth]
- [Firebase Realtime Database]
- [Facebook SDK]
